# dc-projects - Doctoral Candidate Project Templates

## Purpose
Templates for designing individual DC research projects that meet RAISE AI-in-science scope requirements. Includes 4 thematic clusters with 15 example project descriptions.

## Key Files
| File | Description |
|------|-------------|
| `DC-Project-Template.md` | Markdown source with scope checklist, project summary table, and 15 DC project examples across 4 clusters |
| `DC-Project-Template.html` | Styled HTML version for GitHub Pages |

## For AI Agents

### Working In This Directory
- Every DC project must have AI as INTEGRAL to the research (not instrumental)
- 4 clusters: Physical Sciences (DC1-4), Life Sciences (DC5-8), Earth/Environmental (DC9-11), Social Sciences/Cross-cutting (DC12-15)
- Each DC needs: research objectives, methodology, AI + domain integration, training plan, supervision, secondments
- Mobility rule: researcher must NOT have resided >12 months in host country in prior 36 months
